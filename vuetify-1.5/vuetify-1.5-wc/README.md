# vuetify-1.5-wc

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve:app
```

### Compiles and minifies web components
```
yarn build:wc
```

### Compiles and serves web components for development
```
yarn build:wc && yarn serve:wc
```

### Lints and fixes files
```
yarn lint
```
